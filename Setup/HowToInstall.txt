1. Разрешить установку приложений, не из официального источника (sideloading)
	Settings > Update & Security > For developers > enable Developer Mode
	или 
	Параметры > Обновление и безопасность > Для разрабочиков > разрешить Режим разработчика 
2. Выполнить скрипт Install.ps
	- нажать на нём правой кнопкой мыши
	- Run with PowerShell (выполнить с помощью PowerShell)
	- На запрос изменения политики выполнения выбрать "Y" (Да), нажать ввод, подождать 
	- После соответствующего запроса - снова нажать ввод
	- Запрос установки сертификата подтвердить выбором "A" (этот шаг установит сертификат разработчика в папку "Trusted people" (оверенные лица)) 
3. Запустить приложение CulinaryCalculator.UWP. В первый запуск оно проинициализирует базу данных и закроется.
4. Запустить приложение CulinaryCalculator.UWP повторно - приложение готово к работе.
